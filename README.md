# grannysmith
A small Python script to check for stock levels of new Apple iPhones

### Installation

The quick way:

```
pip install grannysmith
grannysmith
```

The recommended way:

```
virtualenv venv
source venv/bin/activate
pip install grannysmith
venv/bin/grannysmith
```
